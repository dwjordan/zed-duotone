# DuoTone for Zed

A port of the [DuoTone](https://marketplace.visualstudio.com/items?itemName=sallar.vscode-duotone-dark) VS Code themes to [Zed](https://zed.dev). Based on the original [DuoTone syntax themes for Atom](https://github.com/simurai/duotone-syntax) by [simurai](https://github.com/simurai), ported to VS Code by [Sallar Kaboli](https://github.com/sallar).

## Themes

### Dark

| Theme | Hue 1 | Hue 2 |
|---|---|---|
| **DuoTone Dark Sky** | purple/lavender | peach |
| **DuoTone Dark Sea** | steel blue | bright teal |
| **DuoTone Dark Space** | periwinkle | orange |
| **DuoTone Dark Forest** | sage green | lime |
| **DuoTone Dark Earth** | terracotta | golden yellow |

### Light

| Theme | Hue 1 | Hue 2 |
|---|---|---|
| **DuoTone Light Sky** | deep purple | burnt orange |
| **DuoTone Light Sea** | navy blue | dark teal |
| **DuoTone Light Space** | deep indigo | dark rust |
| **DuoTone Light Forest** | dark forest green | olive |
| **DuoTone Light Earth** | deep terracotta | dark amber |

## Installation

Copy `themes/duotone-dark.json` and `themes/duotone-light.json` to `~/.config/zed/themes/`, then open the theme selector (`cmd+shift+p` → "theme selector: toggle") and search for **DuoTone**.
